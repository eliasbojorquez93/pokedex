# Pokedex

Pokedex is a POC in React Native for search Pokemons.

## Requeriments
- In your phone, active [Developer Mode](https://www.unocero.com/como-se-hace/como-activar-y-desactivar-las-opciones-de-desarrollador-en-cualquier-telefono-android/)
- Active USB Debugging.
- connect your phone by usb.
- accept usb debugging permissions

## Installing dependencies

you need to follow the steps of the [React Native](https://reactnative.dev/docs/environment-setup) guide:

- Development OS: Windows
- Target OS: Android
- Download [Android Studio](https://developer.android.com/studio)
- [Setting up the development environment](https://reactnative.dev/docs/environment-setup)
- Clone repo: https://github.com/eliasbojorquez93/pokedex.git
- Go to repo folder:
```bash
cd pokedex
```

## Installation

In repo folder, change branch.

```bash
git checkout develop
```

Use the package manager npm or yarn.

```bash
npm i or npm install
```

in terminal runs the following:
```bash
npm start or react-native start
```
in another terminal (do not close the previous one):
```bash
npm run android or react-native run-android
```
